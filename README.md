# 🎮 Rock-Paper-Scissors Game

A simple Python implementation of the classic **Rock-Paper-Scissors** game where you play against the computer.  
The computer makes random choices, and the game keeps track of wins until you decide to quit.

---

## 🚀 Features
- Play against the computer with random choices.
- Tracks the number of wins for both player and computer.
- Quit anytime by typing `Q`.
- Declares the overall winner at the end.
